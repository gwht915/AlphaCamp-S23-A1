5.	Installing – 專案安裝流程
	打開terminal, Clone此專案至本機
	git clone https://github.com/gwht915/AlphaCamp-S23-A1.git
	進入存放此專案的資料夾
	cd restaurant_list
	安裝套件
	$npm install -g nodemon
	啟動伺服器，執行app.js專案
	$nodemon app.js 或
	$npm run dev
	在瀏覽器中輸入 http://localhost:3000
